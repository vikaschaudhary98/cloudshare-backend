#SOLUTION
Pass by Value means that when an argument is passed to a function, a copy of the actual data is passed. Therefore, any changes made to the parameters inside the function do not affect the actual data outside the function

Pass by Reference means that when an argument is passed to a function, the memory address of the actual variable is passed, not a copy. Therefore, any changes made to the parameter inside the function directly modify the original data.

Call by Value is the same as Pass by Value. The term "call" refers to how the function is invoked, and the "by value" refers to how the data is passed. When a function is called by value, a copy of the argument is passed to the function.

Call by Reference is the same as Pass by Reference. The function receives a reference (address) to the actual argument, allowing the function to modify the original data.
